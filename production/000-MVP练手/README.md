# 000 · MVP练手 · 成片素材目录

脚本：[episodes/000-MVP练手.md](../../episodes/000-MVP练手.md)

导演预览页：[storyboard.html](storyboard.html)

这是第一周练手片，目标不是做炫技镜头，而是用 3 镜跑通「脚本 → 静帧 → 少量 i2v → 声音 → 剪映导出」闭环。

素材统一按场镜号命名，例如 `1场1镜.png`、`1场1镜.mp4`。

| 目录 | 内容 |
|------|------|
| `stills/` | 静帧、手机通知截图、角色种子图 |
| `clips/` | Runway 单镜视频、剪映导出的单镜片段 |
| `audio/` | 旁白、音效、BGM、字幕 cue |
| `export/` | 草稿与最终 mp4 |

## 本期 i2v 决策

| 场镜 | 做法 | 原因 |
|------|------|------|
| `1场1镜` | 静帧 + 剪映 105% 慢推 | 交代深夜办公室，不需要复杂动作 |
| `1场2镜` | MJ 静帧 + Runway 轻微推近 | 唯一需要人物表情变化的镜头 |
| `1场3镜` | 手机截图/剪映通知动画 + 切黑 | 中文必须清晰，0 成本完成反转 |

## 推荐制作顺序

1. 先做 `stills/1场1镜.png` 和 `stills/1场2镜.png`。
2. 只把 `1场2镜` 丢进 Runway 做 5 秒轻微推近。
3. 用剪映制作 `1场3镜` 手机通知下滑动画。
4. 按 `audio/captions.txt` 手工对齐字幕和台词。
5. 对照 `checklist.md` 通过后导出 `export/000-MVP练手-final.mp4`。
