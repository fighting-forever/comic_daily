素材统一按场镜号命名，例如 `1场1镜.png`、`1场1镜.mp4`。目录不再按场拆分，便于剪映、Runway 和人工检查直接批量处理。

| 目录 | 内容 |
|------|------|
| `stills/` | 静帧、手机截图、角色种子图 |
| `clips/` | Runway 单镜视频、剪映导出的单镜片段 |
| `audio/` | 旁白、音效、BGM |
| `export/` | 草稿与最终 mp4 |
