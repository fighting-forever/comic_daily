## 新增一篇

1. 复制根目录下的 [TEMPLATE.md](TEMPLATE.md)。
2. 将副本保存到 `episodes/`，文件名：`{三位序号}-{短标题}.md`（例如 `002-项目名称.md`）。
3. 按模板填写元信息、角色、场景与分镜。
4. 在本 README 的 **连载目录** 表格中追加一行。

## 目录说明

| 路径 | 说明 |
|------|------|
| [TEMPLATE.md](TEMPLATE.md) | 新篇脚本模板，复制即用。 |
| [episodes/](episodes/) | 各期脚本存放目录。 |
